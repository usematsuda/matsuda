 About NyARToolKit for Android SDK
----------------------------------------------------------------------
 * NyARToolKit の Android SDK 版です
 * Android SDK 2.2 platform(API level 8) 以上で動作確認を行なっています
 * A虎＠(nyatla)氏の NyARToolkit for Java v4.1.1 を基盤に作られています
 * Processingライクなスケッチシステムを搭載しています。高速なプロトタイピングができます。
 * 処理系の最適化により、3.0系と比較して大幅に高速化しています。
